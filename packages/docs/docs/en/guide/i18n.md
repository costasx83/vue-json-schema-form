# Multilingual Support

## Default Chinese
Default language is Chinese, including form content multilingual and ajv validation information.

## Configure Multilingual

### Form Content Multilingual

* Configure default `form-footer` confirm and cancel button

```js
formFooter = {
    okBtn: 'Save', // Confirm button text
    cancelBtn: 'Cancel' // Cancel button text
}
```

* Configure form rendering multilingual text

Directly configure through the corresponding schema. Also need to use your project's own translation method to implement.

:::tip
Here it is recommended to configure all multilingual text through `ui-schema`

This ensures that `schema` remains standard JSON Schema syntax, making it easy to quickly convert for backend use when backend unified validation is needed.
:::

For example:

::: demo
```html
<template>
    <vue-form
        v-model="formData"
        :schema="schema"
        :form-footer="formFooter"
        :ui-schema="uiSchema"
    >
    </vue-form>
</template>
<script>
// Mock language pack
const transKeyMap = {
    userName: 'Username'
};

// Mock translation method
const trans = key => transKeyMap[key] || `transKeyMap.${key}`;

export default {
    name: 'Demo',
    data() {
        return {
            formData: {},
            schema: {
                type: 'object',
                required: [
                    'userName'
                ],
                properties: {
                    userName: {
                        type: 'string',
                        title: 'Username'
                    }
                }
            },
            formFooter: {
                okBtn: 'Save', // Confirm button text
                cancelBtn: 'Cancel' // Cancel button text
            },
            uiSchema: {
                userName: {
                    'ui:options': {
                        title: trans('userName')
                    },
                }
            },
        };
    }
};
</script>
```
:::

### Default Form Validation Info Multilingual
Schema validation uses `ajv`, `i18n` directly uses [ajv i18n](https://github.com/ajv-validator/ajv-i18n)

```ssh
npm install ajv-i18n
```

Import the language you need in your project:
```js
import localizeEn from 'ajv-i18n/localize/en';
import { i18n } from '@lljj/vue-json-schema-form';

// Set language to use
i18n.useLocal(localizeEn);
```
